Connexion à l'API geo.gouv

Test de connexion avec l'API Géo du gouvernement : retrouver une ville à partir de son code postal, voir site https://api.gouv.fr/les-api/api-geo

Lien de connexion à l'API : https://geo.api.gouv.fr/communes?codePostal=CODE_POSTAL&fields=,nom,code,codesPostaux,centre,surface,codeDepartement,departement,codeRegion,region,population&format=json&geometry=centre
